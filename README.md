# Бронь · агрегатор заведений

В архиве лежат прототипы веб-сайта и мобильного приложения для сервиса бронирования заведений.

## Как открыть

- `index.html` — основная веб-версия агрегатора без декоративной рамки браузера.
- `mobile.html` — мобильное приложение: витрина экранов и двух визуальных вариантов.
- `web.html` — исходный десктопный макет внутри рамки браузера.
- `Бронь - десктоп (standalone).html` — автономная упакованная версия десктопного макета.

## Из чего собрано

- `app/phone.jsx` — общие иконки, данные заведений и мобильные UI-элементы.
- `app/web-shell.jsx` — веб-навигация, сайдбар, верхняя панель и базовые элементы.
- `app/web-pages.jsx` — экраны веб-версии: список заведений, карточка, брони, профиль.
- `colors_and_type.css` — цвета, типографика и дизайн-токены.

Файлы можно открывать прямо в браузере, но для JSX-прототипов нужен доступ к CDN React/Babel.
